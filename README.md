# Selenium Signup Automation (Data-Driven Testing)

This project demonstrates data-driven automation testing of a Sign-Up functionality
using Selenium WebDriver and Python.

## Features
- Automated Sign-Up form validation
- Data-driven testing using Excel
- Tested with valid and invalid inputs
- UI validation for Create Account button enable/disable state

## Tools & Technologies
- Python
- Selenium WebDriver
- Excel (xlrd)

## How to Run
1. Install required dependencies
2. Update test data in Excel file
3. Run the Python script

## Note
This project is created for learning and practice purposes.
